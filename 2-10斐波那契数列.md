多种实现方法,尽量不要使用递归实现(效率很低且容易造成栈溢出)  
**普通方式，只输出最后一个数**  
```
class Solution(object):
    def fib(self, N):
        """
        :type N: int
        :rtype: int
        """
        a=0
        b=1
        if N<=0:
            return a
        else:
            for i in range(N-1):
                a,b=b,a+b
            return b
```
